== Description ==

Order posts search results by post_title relevance in Advanced Custom Fields in fields of types Post Object, Relationship or Page Link.


== Installation ==

1. Upload `ACF admin improve search post results` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In Custom Fields options, locate the field settings of Post Object, Relationship or Page Link type and enable option to ’sort results by most relevant post_title’
